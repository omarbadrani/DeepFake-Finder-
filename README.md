# DeepFake-Finder-
Application avancée combinant deep learning et analyse d'images pour identifier les créations IA (DALL-E, Midjourney, Stable Diffusion). Interface intuitive avec analyse par lots, visualisations détaillées et entraînement personnalisé. Outil essentiel pour photographes, journalistes et créateurs souhaitant vérifier l'authenticité des images.
